Instalar Node JS 

Ejecutar en terminal los siguientes comandos



```
$ git clone https://github.com/boscotron/JMY3Bannana-Little.git
$ cd JMY3Bannana-Little
$ npm install
$ ionic serve

```

ios
```
$ cordova add platform ios

$ ionic build ios

```

Android
```
$ cordova add platform android

$ ionic build android

```
